<h3>Nginx/Node</h3>
  <ol>
    <li>Com o repositório completo clonado crie uma pasta de nome <b>mysql</b> dentro de "/full_cycle/docker/node_nginx";</li>
    <li>Execute o comando <code>docker-compose build</code> no mesmo diretório em que se encontra o arquivo <b>docker-compose.yaml;</b>></li>
    <li>Em seguida execute o comando <code>docker-compose up -d</code></li>
    <li>Abra o navegador e acesse o endereço <b>http://localhost:3000</b> para visualizar o resultado</li>
  </ol>
